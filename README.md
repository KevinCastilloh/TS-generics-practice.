# TypeScript Inventory Management System

A practical demonstration of **TypeScript Generics** and type safety. 

## 🧠 What I practiced here:
- **Generics (`<T>`)**: I implemented a flexible `InventoryItem` interface that can handle different data types (Books, Electronics, etc.) while maintaining strict typing.
- **Type Guards**: Used the `in` operator to check properties at runtime and ensure safe data handling.
- **Data Structures**: Created functions to manage and search through collections using modern JS methods like `.find()`.
